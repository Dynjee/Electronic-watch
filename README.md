# Electronic-watch w/ Schematic

<img src="images/DSC00274.JPG" width="800">

The electronic watch is built using the following components:

Microcontroller (main controller for time logic)
32.768 kHz crystal oscillator (time reference clock)
CR2032 battery (power source)
Battery holder
LED display
Momentary push button (4.5 mm)
1 kΩ resistor (current limiting)
104 ceramic capacitor (decoupling/filtering)
22 pF ceramic capacitor (used with crystal oscillator)
