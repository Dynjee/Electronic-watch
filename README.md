<h2>Components Used</h2>

<ul>
  <li>Microcontroller (main controller for time logic)</li>
  <li>32.768 kHz crystal oscillator (time reference clock)</li>
  <li>CR2032 battery (power source)</li>
  <li>Battery holder</li>
  <li>LED display</li>
  <li>Momentary push button (4.5 mm)</li>
  <li>1 kΩ resistor (current limiting)</li>
  <li>104 ceramic capacitor (power noise filtering / decoupling)</li>
  <li>22 pF ceramic capacitors (used with crystal oscillator stability)</li>
</ul>

<h2>Purpose of Key Components</h2>

<ul>
  <li><b>Crystal oscillator:</b> provides accurate timing reference (32.768 kHz standard for clocks)</li>
  <li><b>Capacitors:</b> stabilize voltage and ensure oscillator accuracy</li>
  <li><b>Resistor:</b> protects LED by limiting current</li>
  <li><b>Button:</b> user input for setting or switching modes</li>
</ul>
